About

I have a strong interest in the intersection of compilers and hardware, and I'm currently working on a project to improve compiler optimizations for GPUs. I'm also interested in programming languages, and I'm currently working on a project to improve the performance of a functional programming language.

I'm well experienced in C, C++, and Python through projects and internships. Prior to Purdue, I graduated from University of Mumbai with a Bachelor's in Computer Engineering and experience in full-stack web development, machine learning, and embedded systems.
